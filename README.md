# Handwritten-Alphabet-Recognition-System
A real-time, deep learning-based Handwritten Alphabet Recognition System designed to interpret and convert handwritten English alphabets (A–Z) into a digital format. Built using a Convolutional Neural Network (CNN) architecture via TensorFlow and integrated with OpenCV, this system achieves a high prediction accuracy of 90%–96%. To overcome the challenges of diverse individual writing styles, variations in stroke thickness, and input noise, the system utilizes a robust automated preprocessing pipeline. Input characters drawn on a digital interface or uploaded as images are dynamically grayscaled, thresholded, isolated using contour detection, and normalized to standard $28 \times 28$ dimensions. The system leverages the EMNIST Letters dataset alongside extensive data augmentation techniques (such as rotation, cropping, and shifting) to guarantee generalization.
